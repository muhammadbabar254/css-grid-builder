# 📐 CSS Grid Builder

A powerful CSS Grid Builder built with HTML, CSS, and JavaScript.  
Create responsive grid layouts visually and generate CSS instantly.

---

## ✨ Features

- 📐 Live grid preview
- 🔢 Custom rows and columns
- 🎨 Visual layout builder
- ⚡ Instant CSS generation
- 📋 One-click CSS copy
- 💻 Beginner friendly tool
- 🚀 Lightweight and fast

---

## 📂 Project Structure

```
css-grid-builder/
│── index.html
│── style.css
│── app.js
│── README.md
```

---

## 🚀 How to Use

1. Open `index.html` in browser
2. Set number of rows and columns
3. Click **Generate Grid**
4. View live layout
5. Click **Copy CSS** to use in your project
